  1. Express.js, 
  2. npm dependencies.
  3. package.JSON
  4. VScode.
  5. GitHub.
  6. licenses and their badges.
  7. MySQL2. 
  8. Sequelize.
  9. dotenv package.
  10. express-session.
  11. connect-session-sequelize.
  12. handlebars.
  13. bulma css.
  14. bycrypt.
  15. deployment with heroku.
